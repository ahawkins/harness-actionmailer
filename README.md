# Harness::ActionMailer

Log all `ActionMailer`'s performance metrics with Harness.

## Installation

Add this line to your application's Gemfile:

    gem 'harness-actionmailer'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install harness-actionmailer
